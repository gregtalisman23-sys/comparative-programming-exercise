# Student Registration Status Checker

## Purpose
This project demonstrates how the same simple program can be implemented in four different programming languages: Python, Java, JavaScript, and C.

## How to Run
- **Python**: `python registration_checker.py`
- **Java**: `javac RegistrationChecker.java && java RegistrationChecker`
- **JavaScript (Node.js)**: `node registration_checker.js`
- **C**: `gcc registration_checker.c -o checker && ./checker`

## What It Demonstrates
- Input handling across languages
- Conditional logic
- Output formatting
- Comparative syntax and structure
